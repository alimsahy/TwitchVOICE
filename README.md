# TwitchVOICE
Open source Twitch chat bot with speech recognition features

# Usage
- Go https://twitchapps.com/tmi/
- Connect with your Twitch account
- Generate password
- Open Program.cs and type your own twitch username and generated oauth password
- RUN!
